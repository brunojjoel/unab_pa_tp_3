# Objetivos a cumplir

Los estudiantes deberán:

1. Entender y comprender ejercicios prácticos
2. Trabajar en grupos
3. Manejar un IDE (cualquiera a elección)
4. Aplicar programación básica en Python 3
5. Comprender conceptos vinculados a la Programación Orientada a Objetos (POO)
6. Utilizar Git y GitHub a nivel básico (crear repositorio y subir archivos)
